
这是一个前端脚手架项目。

### 使用方法
```bash
$ npm install tplhk-cli -g # 安装
$ npm update tplhk-cli -g  # 更新
$ tpl -v                   # 显示当前版本
  version is 0.0.2
$ tpl create project_name  # 创建项目
```
